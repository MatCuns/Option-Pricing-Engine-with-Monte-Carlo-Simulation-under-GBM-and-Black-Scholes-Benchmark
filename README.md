# Option-Pricing-Engine-with-Monte-Carlo-Simulation-under-GBM-and-Black-Scholes-Benchmark
C++ implementation of a Monte Carlo engine to price European call and put options under the Geometric Brownian Motion assumption. The project simulates asset price paths, computes discounted payoffs, and compares the Monte Carlo estimates with the analytical Black–Scholes solution to validate accuracy and convergence.
